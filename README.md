# 🫧 Bubble Rush

A fast-paced bubble-popping web game built with **React** and **Tailwind CSS**.

## How to Play

- Click **Start Game** to begin a 30-second round
- Bubbles appear across the screen — click them as fast as you can to pop them!
- Each popped bubble adds **1 point** to your score
- When time runs out, see your final score and play again

## Features

- Colorful bubbles spawning at random positions and sizes
- Live score counter and countdown timer
- Timer pulses red when 5 seconds remain
- Pop animation on each bubble
- Results screen with final score and "Play Again" button

## Development

